Tech Lead Onboarding Project

Tech leads should be proficient with a wide variety of web development tools including

  • Frontend Development: React, JavaScript/TypeScript, CSS, JSX/TSX
  • Backend Servers: Runtimes, APIs, Basic Networking (HTTP)
  • DBMS: SQL v. NoSQL, Schemas, Query Builders v. ORMs
  • Authentication
  • Version Control: Git, GitHub
  • Deployment

It might be a little while since you've touched full stack web dev and we do not expect you to be an expert with all of these skills. We've developed a small project covering a full stack system to refresh some of your technical skills.

We've opted to do a more practical project instead of a "traditional" onboarding with slideshows and synchonous meetings. We hope this is far more interesting and helpful than sitting through Zoom calls! :)

Functionality Requirements

  • Build a user interface with React to display messages in a conversation and send messages to a chatbot.
    • The prompt sent to OpenAI must include ALL chat messages in a conversation. Previous messages must be sent as context to maintain a fluid conversation.
  • Log all prompts from users and responses from OpenAI in a non-volatile memory system like a database.
  • Add system prompts to refine the behavior of your chatbot. Is it a bot that responds with JavaScript related puns? Or a bot interested in becoming a Terminator? It's totally up to you!
      How do I send system prompts and structure conversations?
  • Deploy your fully functional site to the internet.

Additional features to implement if you need practice with these areas of web dev:

  • Implement a basic (but secure) authentication system to track user chat messages.

Deadlines and Check-ins

You will be given approximately 4 weeks to complete the project. We expect this will to take between 4-10 hours to complete depending on your experience level. It should be more than manageable if you spend a couple hours per week on it. Please reach out ASAP if you're spending significantly more time on the project than our estimate.

We'll have short (~15 min) weekly check-ins to answer any questions that have come up about the onboarding project.

Please send Gabriel and Ben your planned tech stack before you start implementing your design! Try to have an initial tech stack planned and sent to us before your first check-in.
